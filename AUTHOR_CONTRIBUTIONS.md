## Rafael Louie Miguel

**Role:** Infrastructure and Backend Engineer

Rafael Louie Miguel led the infrastructure and backend setup of the project and played a major role
in making the application deployable as a complete working platform. This included setting up the
server deployment, preparing the production environment, configuring the supporting services, and
ensuring that the web application and backend could run together in a reliable way. His work helped
turn the project from a development codebase into something that could be properly hosted and
maintained.

He also contributed to the backend content setup and the operational side of the system. This
included helping organize the PocketBase environment, configuring Caddy, documenting the setup
process, and improving how the application is exposed in production for better control and security.
He also set up the Cloudflare-based security layer used to help protect the application from DDoS
and bot attacks. In addition, Rafael Louie Miguel helped guide the project toward a cleaner
from-scratch setup, including the deployment flow for the SvelteKit app and the supporting backend
structure behind it.
